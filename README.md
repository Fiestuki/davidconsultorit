## ✨ Características Principales

### 1. Diseño "Cyber-Tech" Profesional
*   **Tema Oscuro (Dark Mode):** Implementado por defecto para reducir la fatiga visual y alinearse con la estética estándar del sector de ciberseguridad.
*   **Glassmorphism:** Uso de transparencias y desenfoques (`backdrop-filter`) en tarjetas y contenedores para una apariencia moderna y profunda.
*   **Paleta de Colores:** Tonos pizarra (`Slate`) combinados con acentos en azul cian neón (`Sky-400`) para destacar elementos clave.

### 2. Experiencia de Usuario (UX)
*   **Efecto Typewriter:** Animación dinámica en el encabezado que rota entre mis roles principales (Admin de Sistemas, Ciberseguridad, Soporte Senior).
*   **Navegación Suave:** Scroll suave entre secciones y menú de navegación fijo con efecto de desenfoque.
*   **Totalmente Responsive:** Adaptación perfecta a dispositivos móviles, tablets y escritorio mediante Media Queries.

### 3. Secciones Destacadas
*   **Hero Section:** Presentación impactante con llamada a la acción clara (Contactar / Descargar CV).
*   **Stack Tecnológico:** Visualización gráfica de habilidades técnicas (Linux, Cisco, Windows Server, etc.).
*   **Línea de Tiempo (Timeline):** Representación visual clara de mi trayectoria profesional en empresas como Inerza, Fujitsu y Consultores Canarios.
*   **Certificaciones:** Tarjetas destacadas para credenciales oficiales de Google y Cisco.

## ⚙️ Instalación y Despliegue Local

Si deseas ejecutar este proyecto en tu entorno local para pruebas o desarrollo:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/tu-repo.git
    cd tu-repo
    ```

2.  **Abre el archivo:**
    Simplemente abre el archivo `index.html` en tu navegador preferido. No se requiere servidor local ni instalación de dependencias (Node/npm) debido al uso de tecnologías nativas.

3.  **Configuración del Formulario:**
    Para que el formulario de contacto funcione, debes registrarte en [Formspree](https://formspree.io/) y reemplazar el atributo `action` en la etiqueta `<form>` dentro de `index.html`:
    ```html
    <form action="https://formspree.io/f/TU_CODIGO_UNICO" method="POST">
    ```
